1) Intermediate Modeling

Move beyond primitives and simple extrusions. Focus on clean topology, controlled edge flow, and detail where it matters.

Edge Flow & Control

  • Use Ctrl + R for loop cuts; maintain quads for deformation.
  • Apply support loops near sharp edges before Subdivision.
  • Avoid long thin triangles; redirect poles with careful cuts.

Sculpting Intro

  • Switch to Sculpt Mode for organic forms.
  • Use Dyntopo or Multires based on needs.
  • Remesh to unify topology before retopology.
Keep non‑destructive workflows: model with modifiers and apply only when necessary.

2) Advanced Modifiers

Combine modifiers to build complexity quickly.

  • Array for repeated patterns (fences, windows).
  • Boolean for cutouts; clean up with bevels or remesh.
  • Shrinkwrap to conform topology to a target surface.
  • Mirror + Weighted Normal for clean shading.
Order matters: Mirror → Bevel → Subdivision is a common stack for hard‑surface models.

Visual: chaining modifiers for non‑destructive modeling.

3) UV Unwrapping & Texturing

Good UVs prevent stretching and make texturing predictable.

  1. Mark seams where the model naturally splits (Ctrl + E).
  2. Unwrap (U) and pack islands; keep texel density consistent.
  3. Bake maps (normal/roughness) if needed; name textures clearly.

Procedural

Use noise, voronoi, and color ramps for tileable, resolution‑independent detail.

Image Textures

Load PBR sets (BaseColor/Metallic/Roughness/Normal) with Node Wrangler (Ctrl + Shift + T).

UDIM Intro

For large assets, split UVs into tiles to increase resolution without huge maps.

Diagram: seams → unwrap → packed UV islands with consistent texel density.

4) Shader Nodes & PBR

Create flexible, reusable materials with the Shader Editor.

  • Principled BSDF is physically‑based and covers most surfaces.
  • Mix shaders for layered looks (e.g., coat, dust, edge wear).
  • Use Ambient Occlusion, Fresnel, and Curvature maps for realism.
Keep values physically plausible: roughness 0–1, metallic only for metals, albedo within 0.02–0.9 reflectance.

Node sketch: PBR texture set plugged into Principled BSDF with mask‑based mix.

5) Lighting with HDRI & Rigs

Shape mood and readability with lighting.

  • Use HDRIs for natural ambient light; rotate for the best angle.
  • Build a 3‑point rig (Key/Fill/Rim) and balance intensities.
  • Control color temperatures for contrast (warm key, cool fill).

Lighting diagram: key/fill/rim arrangement around a subject.

6) Cycles Rendering (Advanced)

Push quality while keeping render times reasonable.

  • Use Adaptive Sampling and Denoising (OpenImageDenoise/OptiX).
  • Set light clamping to reduce fireflies; enable MIS for emissive surfaces.
  • Render at 16‑bit PNG/EXR for color grading; manage color with Filmic.

7) Rigging Basics

Add an armature and deform a mesh correctly.

  1. Create an armature; align bones to the mesh.
  2. Parent mesh to armature with automatic weights (Ctrl + P).
  3. Fix weights in Weight Paint; add IK constraints to limbs.
Name bones clearly (e.g., upper_arm.L) and mirror weights for symmetry.

Sketch: armature hierarchy and IK handle on the leg.

8) Animation: Walk Cycle & Camera

Create a basic loop and present it cinematically.

  • Block key poses (contact, down, passing, up) on the timeline.
  • Refine in the Graph Editor; add overlap and offset.
  • Animate a camera path; use depth of field for focus.

Pose sheet: contact → passing → up → loop with camera dolly.

9) Physics Simulations

Add believable motion with built‑in solvers.

Rigid Body

Stack boxes and drop a sphere; tweak mass and friction.

Cloth

Pin vertices, increase quality steps, and collide with a moving object.

Fluid (Intro)

Use Mantaflow for basic pours; cache at low resolution first.

10) Intermediate Project: Stylized Character Bust

Combine modeling, sculpting, texturing, lighting, and rendering into a single polished piece.

  1. Block primary forms with simple meshes and sculpt for shapes.
  2. Retopologize major areas; unwrap UVs and create simple PBR materials.
  3. Light with HDRI + rim; render in Cycles with denoise; output 2K portrait.

Showcase ready! Publish on ArtStation/Behance and keep source files organized for iteration.

11) Optimization & Export

  • Apply scale/rotation (Ctrl + A) before export.
  • Pack textures, triangulate if needed for games, and check scale units (meters).
  • Export FBX or GLTF/GLB for engines; test import in a blank scene.

Frequently Asked Questions

Do I need a tablet for sculpting?

It helps with pressure sensitivity and finer control, but a mouse works for this course.

Which renderer should I use?

Use Eevee during look‑dev for speed and Cycles for final frames when realism is important.

How big should my textures be?

Use 2K–4K for hero assets viewed up close; smaller (512–1K) for background props.

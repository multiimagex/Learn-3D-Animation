1) Install Blender

Theory

Blender is a free, open‑source 3D creation suite. The installer includes modeling, sculpting, animation, rendering (Eevee/Cycles), compositing and video editing — everything you need to begin.

Practical

  1. Download the latest stable release for your OS.
  2. Install and launch Blender. On first run, choose Left Click Select and Industry Compatible keymap (optional).
  3. Save preferences: Edit → Preferences → Save Preferences.
Enable Node Wrangler add‑on in Preferences → Add‑ons; it speeds up material setup.

2) Interface Tour

Theory

Blender’s UI is split into Editors: 3D Viewport, Outliner, Properties, Timeline, Shader Editor, etc. You can split/merge areas and save a custom layout as the default.

Practical

  • Identify: Outliner (scene objects), Properties (object and render settings), Timeline (animation).
  • Split an area by dragging corners; change editor type from the top‑left icon.
  • Save startup file: File → Defaults → Save Startup File.
Workspaces (Layout, Modeling, Shading, Animation, Rendering) are presets for common tasks.

3) View & Navigation

Theory

Efficient navigation is key. Orbit, pan, and zoom without breaking focus. Use orthographic views for precision and isolate objects to reduce clutter.

Practical

  • Orbit: MMB drag • Pan: Shift + MMB • Zoom: Ctrl + MMB.
  • Focus selected: Numpad . • Toggle ortho/persp: Numpad 5.
  • Front/Right/Top: 1/3/7 (add Ctrl for opposite).
Use Local View / to isolate the selected object.

4) Objects & Transforms

Theory

Everything in the scene is an object with transform data: location, rotation, scale. Learn to add primitives, edit origins, and reset transforms.

Practical

  1. Add a mesh: Shift + A → Mesh → Cube.
  2. Move/Rotate/Scale: G/R/S • Axis constrain: X, Y, Z.
  3. Snap: Shift + Tab • Reset: Alt + G/R/S.
Apply transforms before exporting: Ctrl + A → Apply All Transforms.

5) Modeling Basics

Theory

Modeling edits the mesh at the component level: vertices, edges and faces. Clean topology (mostly quads) deforms and shades better. Start simple, add detail where the silhouette needs it.

Practical

  • Enter Edit Mode: Tab. Select 1/2/3 for Vertex/Edge/Face.
  • Extrude: E • Inset: I • Loop Cut: Ctrl + R • Bevel: Ctrl + B.
  • Merge: M • Knife: K • Proportional Edit: O.
Keep scale uniform. Avoid long thin triangles and n‑gons in curved areas.

6) Modifiers Intro

Theory

Modifiers are non‑destructive operations that procedurally change your mesh. Common beginner modifiers: Subdivision Surface (smooth detail), Mirror (symmetry), Array (repetition), Bevel (rounded edges).

Practical

  1. Add a Mirror modifier on the X axis. Model only half of the object.
  2. Add Subdivision Surface and control sharpness with support loops.
  3. Use Array to duplicate evenly spaced parts.
Apply modifiers only when necessary. Keep a copy of the non‑applied version.

7) Materials & Shading

Theory

Materials describe how surfaces react to light. The Principled BSDF shader follows PBR rules to produce believable results with a few controls: Base Color, Roughness, Metallic, Normal.

Practical

  • Switch to the Shading workspace. Create a new material.
  • Plug a Noise Texture into Base Color via a ColorRamp for a simple procedural pattern.
  • Use Node Wrangler (Ctrl + Shift + T) to add PBR image textures when available.
Keep albedo values realistic; avoid pure black/white for physically plausible results.

8) Lighting Basics

Theory

Lighting shapes mood and forms. A classic three‑point setup uses a Key (main direction), Fill (softens shadows) and Rim (separates subject from background).

Practical

  1. Add an Area light (Key) from the front‑side; set power and size.
  2. Add a weaker Fill from the opposite side.
  3. Add a Rim behind the subject to outline edges.
For natural ambience, add an HDRI in World settings. Rotate for best highlights.

9) Rendering Basics

Theory

Rendering converts your 3D scene into a 2D image. Eevee is real‑time (fast look‑dev), Cycles is path‑traced (more realistic). Use Filmic color management for natural highlights.

Practical

  • Set output size (1920×1080) and file format (PNG) in Render Properties.
  • For Cycles, set samples (e.g., 64) and enable denoising.
  • Press F12 to render and Image → Save As.
Use render region (Ctrl + B) to test small areas quickly.

10) Animation Basics

Theory

Animation creates motion by changing values over time. Keyframes record values, interpolation blends them, and the Graph Editor refines easing and overshoot.

Practical

  1. Set timeline to 1–120. Select an object and insert a keyframe for Location at frame 1 (I).
  2. Move to frame 60, move the object on X, insert another keyframe.
  3. Open Graph Editor, select curves and set interpolation to Bezier for smooth motion.
Use Auto Keying for faster blocking; disable when not needed.

11) Mini Project: Simple Product Render

Goal

Create a clean product‑style render (a perfume bottle or stylized cup) using primitives, modifiers, materials, three‑point lighting and Cycles rendering.

Steps

  1. Block shapes with Cube/Cylinder; use Bevel for soft edges.
  2. Add Mirror if symmetric; use Subdivision for smoothness.
  3. Create a simple PBR material; add a subtle roughness gradient.
  4. Build a three‑point light rig; adjust shadows for clarity.
  5. Render in Cycles at 1080p with denoise enabled.

Deliverable: One 1920×1080 PNG with a neutral background and good lighting. Name files clearly and keep your .blend organized.

Frequently Asked Questions

What version of Blender should I use?

Use the latest stable release for general learning. If you follow studio add‑ons or specific tutorials, match their version.

How can I make Blender run faster?

Lower viewport samples, disable overlays, use Solid view for modeling, and reduce subdivision levels when not rendering.

Where should I save my files?

Create a project folder with subfolders: /blend, /textures, /renders, /refs. Name versions like product_v01.blend.
